# System operations handler with tauri v1.0.0beta

The application is able to (restart, turning off, sleep mode) the computer with a timer

## Features with timer:
- [x] Restart
- [x] Turning off
- [x] Sleeping mode

### Start Tauri Development Window

```sh
npm run tauri dev
```

### Bundle application with Tauri

```sh
npm run tauri build
```
